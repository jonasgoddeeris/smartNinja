Homework 1.1: FizzBuzz

  • At the top of the JS file declare the number that your program will need to count to.
  • FizzBuzz program starts to count to that number (it prints all the counted numbers in the console).
  • In case the number is divisible with 3, it prints "Fizz" instead of the number.
  • If the number is divisible with 5, it prints "Buzz".
  • If it's divisible with both 3 and 5, it prints "FizzBuzz".
